# Heart Disease Prediction using Neural Networks

## Overview
This project demonstrates how activation functions affect the performance of neural networks in a healthcare prediction task.

The model is trained on a heart failure prediction dataset and compares three activation functions:
- Sigmoid
- Tanh
- ReLU

## Dataset
The dataset was obtained from Kaggle and contains clinical features such as age, cholesterol, and blood pressure.

## Method
A neural network was implemented using Keras with TensorFlow backend. The model includes:
- Input layer
- Two hidden layers
- Output layer for binary classification

Categorical variables were converted using one-hot encoding.

## Results
ReLU achieved the highest accuracy and fastest convergence, demonstrating its effectiveness compared to sigmoid and tanh.
